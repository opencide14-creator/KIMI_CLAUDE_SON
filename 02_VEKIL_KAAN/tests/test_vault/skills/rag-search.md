# RAG Semantic Search

How to perform semantic search against the [[Memory Substrate]] using ChromaDB.

## Overview

The RAG environment is the world both agents inhabit.
Semantic search enables agents to retrieve relevant context before reasoning.

## Query Syntax

Use rag_search with a natural language query. ChromaDB returns the most
similar chunks based on vector similarity. Always retrieve context before acting.
