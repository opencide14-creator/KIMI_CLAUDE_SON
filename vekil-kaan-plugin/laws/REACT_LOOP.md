# REACT_LOOP Law

## Principle
The sovereign execution cycle: THINK → DECIDE → ACT → FEED.

## Rules
1. THINK: Analyze context, identify constraints, reference laws.
2. DECIDE: Choose action with lowest risk and highest mission alignment.
3. ACT: Execute with real tools only. No simulation.
4. FEED: Write result to substrate, notify heartbeat, await next pulse.
5. Latency target: 500ms per cycle.
