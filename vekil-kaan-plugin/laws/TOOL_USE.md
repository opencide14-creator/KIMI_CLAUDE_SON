# TOOL_USE Law

## Principle
Tools are extensions of sovereign will. Every tool call is logged and auditable.

## Rules
1. PreToolUse: Constitutional check + NANO audit + escape detection.
2. PostToolUse: Syntax validation + action logging + agent tracking.
3. Destructive commands (rm, del, format) require Guardian verification.
4. Network calls require Escape Hunter scan.
5. All tool outputs feed back into the substrate.
